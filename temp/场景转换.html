Cocos2d-html5之场景转换

2014-03-03
undefined undefined

建立第一个场景

var StartScene = cc.Scene.extend({
    onEnter: function () {
        this._super();

        //此处的StartSceneLayer为自定义的Layer
        var layer = new StartSceneLayer();
        layer.init();
        this.addChild(layer);
    }
});

加载图片、声音等资源文件，加载完成后执行回调函数载入场景

cc.Loader.preload(res.loadList, function () {
    cc.Director.getInstance().runWithScene(new this.startScene());
}, this);

runWithScene接受一个Scene的实例

上面的代码根本没有使用任何过渡的方法，比较简单地完成了任务。但是在实际的需求中会有这样那样的场景切换的操作，例如：html5游戏中回城的场景切换、死亡时的虚幻等等。Cocos2d-html5引擎中提供了切换场景的一些方法，这些场景切换类都继承自TransitionScene。下面列举一些：

  • cc.TransitionCrossFade.create(t, scene) // 交叉消失两个场景使用cc.RenderTexture对象。
  • cc.TransitionFade.create(t,scene,color) // 淡出即将离任的场景,然后消失在传入的场景。
  • cc.TransitionJumpZoom.create(t, scene) // 跳跃式替换，场景缩小，再加载进来
  • cc.TransitionMoveInB.create(t,scene) // 创建一个在底部，覆盖当前场景
  • cc.TransitionPageTurn.create(t,scene, backwards) // 前翻页式场景替换
  • cc.TransitionRadialCW.create(t,scene) // 逆时针切入
  • cc.TransitionRotoZoom.create(t,scene) // 转换角度替换

上面的参数t为动画执行时间，以秒为单位；参数scene是一个Scene的实例。Cocos2d-html5提供了很多其他样式的Transition类，具体可以上Cocos2d-html5 API文档，搜索关键词transition，下面给出参考的集成体系图。
